# Dashboard de anuncios de coches

Esta aplicación web permite explorar un conjunto de datos de anuncios de venta de coches en Estados Unidos.

## Funcionalidad

- Visualización de la distribución del kilometraje mediante histograma.
- Visualización de la relación entre precio y kilometraje mediante gráfico de dispersión.
- Interfaz interactiva construida con Streamlit.
- Gráficos interactivos creados con Plotly Express.

## Tecnologías utilizadas

- Python
- Pandas
- Plotly Express
- Streamlit
